# Stakeholder-Driven Design of a Blockchain-Based Information System for Real Estate Transactions

This project looks at how blockchain technology could be used to improve real estate transactions, which are often slow and involve many intermediaries. These steps can introduce inefficiencies and sometimes increase the risk of fraud or mistrust between parties.

To understand the problem better, we reviewed existing research on blockchain applications in real estate. In addition, we spoke with real estate agents and people involved in different stages of the transaction process. This helped us understand how the current system works in practice and where the main issues are.

Based on this, we designed a simple model of a real estate transaction system using blockchain concepts. The idea was to represent properties as digital assets and use smart contracts to manage transactions in a more transparent way.

We also implemented basic algorithms to simulate how transactions would work in this system. The goal was not to build a production-ready system, but to explore whether blockchain could realistically improve the process.

Overall, the project shows that blockchain has potential in this area, but there are still many practical and regulatory challenges that need to be solved before it can be widely used.

# Sample Hardhat Project

This project demonstrates a basic Hardhat use case. It comes with a sample contract, a test for that contract, and a script that deploys that contract.

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
